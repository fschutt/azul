# Azul — Ada

⊘ **Toolchain unavailable on macOS-aarch64.** Codegen produces Ada
bindings but `gnatmake` / GNAT-FSF isn't cleanly installable via brew;
the AdaCore GNAT-CE bundle is a separate manual download. The
generated files are kept for documentation purposes and porting to
Linux/Windows where GNAT is available.

See `memory/codegen_rehab_status.md` for the full toolchain matrix.
