### Arc 3 — WB1.1 remill subprocess pipeline (1 commit)

`dll/src/web/transpiler_remill.rs` (`18706f526`):

  - Replaces `todo!()` scaffolding with a real three-stage subprocess pipeline.
  - `lift_function(name, addr, size)` → reads bytes from .text, hex-encodes, calls `remill-lift-17` → `llc -mtriple=wasm32 -filetype=obj` → `wasm-ld --no-entry --export=sub_<addr>`. Returns `WasmModule` with content hash + exports.
  - `lift_and_link_framework(funcs)` → per-function lift loop, single `wasm-ld` batch invocation.
  - Toolchain discovery is layered (env-var → known paths). `is_available()` returns true only when all three binaries are discoverable; lift methods short-circuit to `TranspileError` otherwise.
  - Validated via the experiment side: `cd experiments/transpile-blueprint && cargo run --release -- --lifter=remill` produces a 230-byte WASM module with `\0asm` magic and `sub_100000000` export.
